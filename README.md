### Installation

> [!TIP]
> if you don't use pnpm, you can replace `pnpm` with `npm` or `yarn` in the following commands.

1. Clone the repo

   ```sh
   git clone url
   ```

2. Navigate to the project directory

   ```sh
    cd project-directory
   ```

3. Install pnpm packages

   ```sh
    pnpm install
   ```

### Usage

1. Start the development server
   ```sh
    pnpm dev
   ```
2. Build the project
   ```sh
    pnpm build
   ```
